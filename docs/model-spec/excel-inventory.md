# Excel-Inventar

Diese oeffentliche Datei enthaelt nur eine anonymisierte Struktur. Proprietaere Original-Excel-Inventare werden lokal als `excel-inventory.local.md` abgelegt und sind per `.gitignore` ausgeschlossen.

## Anonymisierte Struktur

- Beispiel-Arbeitsmappe A: sichtbares Blatt `Annahmen`, optionale Ergebnisblaetter.
- Beispiel-Arbeitsmappe B: sichtbares Blatt `Annahmen`, optionale Ergebnisblaetter.
- Originale Blattnamen, Zellwerte, Formeln und Dateinamen werden nicht im oeffentlichen Repository veroeffentlicht.
