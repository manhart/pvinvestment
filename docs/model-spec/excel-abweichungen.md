# Excel-Abweichungen

Excel-Dateien sind Referenzmaterial, aber keine Produktiv-Rechenengine.

Bekannte fachliche Abweichungsgruende:

- keine doppelte Kaufnebenkostenverwertung
- konsistente Zinsbehandlung
- separater AfA-Beginn und Ertragsbeginn
- explizite `sharing_base` fuer Batterie-Profit-Sharing
- parametrisierbares Steuer-Cashflow-Timing
- frei eingebbares Sparplan-Startkapital

Konkrete projektbezogene Excel-Abweichungen werden nicht im oeffentlichen Repository abgelegt.
